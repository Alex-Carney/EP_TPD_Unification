Copy and paste the databases into this folder for use with the rest of the program.

For your conveinence, we include the analyzed data in `.csv` format as well. However, to ensure reproducibility, we recommend using the `.db` files with the provided codebase.

Note that the original raw experimental data cannot realistically be stored in `.csv` format due to the sheer size of the files. Please contact the corresponding or first author for more information.
